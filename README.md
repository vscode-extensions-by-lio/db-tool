# DBTool - VS Code PostgreSQL Explorer

一个轻量级 PostgreSQL 可视化管理插件，在 VS Code 左侧栏显示数据库连接、schema 和表，并可在主窗口打开表格数据进行查看。

Installation From VS Code Marketplace or manually:
`code --install-extension excel-to-sql.vsix`

Development
submod project:
```cmd
npm install
npm run build
```
main project
```cmd
npm install
npm run compile
vsce package
```
